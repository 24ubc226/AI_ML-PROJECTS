# Customer Churn Modelling

## Project Overview
This project predicts whether a customer is likely to leave a company (churn) based on customer data.

## Objective
To identify customers who are at risk of leaving and help businesses improve customer retention.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Algorithm
- Logistic Regression
- Random Forest
- Decision Tree

## Features
- Data preprocessing
- Feature selection
- Model training
- Model evaluation
- Churn prediction

## Dataset
Customer demographic and banking information.

## Results
The model predicts customer churn with good accuracy and helps identify high-risk customers.
